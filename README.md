## ☁️ 云策略台账系统

> 📄 文件：[policy-ledger.html](policy-ledger.html)
> 🌐 在线访问：[GitHub Pages](https://wanghahaniu.github.io/my-second-project/policy-ledger.html)

一个用于记录、分类和分析群体发言策略的在线工具。

### ✨ 功能特性

- 📝 **记录发言** - 记录发言人、内容、日期和策略分类
- 📊 **实时统计** - 显示发言数、参与人数、策略数量
- 👥 **按人员分类** - 查看每个人提出了哪些策略建议
- 📋 **按策略分类** - 查看每个策略由哪些人提出
- ☁️ **本地存储** - 使用浏览器 localStorage
- 📱 **响应式设计** - 支持各种设备尺寸

### 📖 使用指南

1. **记录发言**：填写发言人、内容、日期和策略分类，点击「➕ 添加发言」
2. **查看统计**：切换标签页查看「所有发言」「按人员分类」「按策略分类」
3. **管理数据**：支持删除单条记录、导出 JSON、清空数据
