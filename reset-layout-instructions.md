# 恢复 VS Code 面板布局说明

以下内容是将 VS Code 界面恢复到默认布局的步骤总结。可以将此文档提供给其他自动化工具或用作参考。

## 1. 重置视图位置
1. 按 `Ctrl + Shift + P` 打开命令面板。
2. 输入 `View: Reset View Locations` 并选择执行。
3. 关闭并重启 VS Code 以确保生效。

> 这一步仅重置原生面板（文件编辑区、资源管理器、终端、输出等），不会影响扩展添加的视图。

## 2. 关闭右侧扩展面板
1. 用鼠标点击右侧面板内部使其获得焦点，或者按 `Ctrl+0` 然后 `→` 切换焦点。
2. 在面板标题的右侧查找一个小的 `×` 关闭按钮，点击即可关闭当前面板。

> 如果看不到 `×`，说明面板不是活动状态，或者它作为活动栏显示，需要通过右键图标隐藏或卸载相关扩展。

## 3. 隐藏/禁用扩展视图
- 若不想看到某个扩展的侧边栏视图，可在扩展视图中右键 → "隐藏视图"。
- 或者在扩展面板中找到对应扩展，选择禁用/卸载。

## 4. 使用快捷键管理布局
- `Ctrl + B`：切换左侧面板显示/隐藏。
- `Ctrl + J`：切换底部面板显示/隐藏。
- `Ctrl + 1`、`Ctrl + 2` 等：切换左侧活动栏选项。
- `Ctrl + K Z`：专注模式。

## 5. 重启 VS Code
所有修改后建议重启，以应用新的布局设置。
